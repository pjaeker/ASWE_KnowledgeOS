# Artifact families

Decide closeout by family, not by isolated file.

## Mandatory families

- RepoStatusUpdate
- active Workstream
- Structural Reference / Policy / ADR / Protocol
- Codex-App Operating Guidance
- ENTRY_LATEST
- NextChat-Handoff

## Notes

`Codex-App Operating Guidance` includes, when relevant:
- start protocols
- topology rules
- thread-local system prompts
- runtime-adapter guidance

Each family must be decided as:
- create
- update
- no-change
