## Output template

1. Work mode
   - triage-only / repo-evidence-only / mixed

2. Session-anchor alignment
   - inherited entry or equivalent anchor
   - inherited status or dual-status view
   - active technical workstream
   - drift or conflict: yes/no

3. Active scope
   - investigation frame, subsystem, or hypothesis family under inspection

4. Key findings
   - concise triage conclusions only

5. Evidence anchors
   - files, outputs, overlays, or signals that support the ranking

6. Confidence
   - low / medium / high with one-sentence reason

7. Open questions
   - unresolved gaps that still block prioritization

8. Recommended next action
   - smallest safe next investigation step

9. Role handoff / next bounded role
   - none / aswe-mapper / aswe-observer / aswe-test-agent / aswe-doc-agent

10. Problem frame
11. Ranked hypotheses
12. Evidence supporting / weakening each path
13. Dead ends to avoid repeating
