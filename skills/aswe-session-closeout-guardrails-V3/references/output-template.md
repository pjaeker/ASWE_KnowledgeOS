# Reference: Output Template

## Closeout checkpoint

1. session summary
2. leading repo evidence used
3. canonical entry after closeout
4. latest status update found
5. status referenced by entry
6. active workstream
7. handoff status
8. contract status
9. next recommended starting focus

## Closeout writingslice

1. PATCH-SPEC
2. FILES TOUCHED
3. PLANNED DIFF
4. artifact decisions by class
5. no-change justifications

## Reporting discipline

Separate hard facts, plausible candidates, provisional inferences, and blockers.
Do not report the closeout as complete if the checkpoint exists but the writingslice decision is still missing.
