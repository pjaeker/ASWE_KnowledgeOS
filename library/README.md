# Library

## Zweck der Schicht
- Sammlung von Quellen, Evidenz und Rohmaterial.
- Nachvollziehbare Basis für spätere Ableitungen.
- Trennung von Rohwissen und kuratierter Darstellung.

## Erlaubte Artefakte
- Quellnotizen und Literaturhinweise.
- Roh-Exzerpte und Evidenzsammlungen.
- Verweise auf Originaldaten/Primärquellen.

## Verbotene Artefakte / Anti-Patterns
- Kuratierte Leitfäden oder finale Anweisungen.
- Unbelegte Behauptungen ohne Quellenbezug.
- Ausführbare Templates ohne Kontextdokumentation.
