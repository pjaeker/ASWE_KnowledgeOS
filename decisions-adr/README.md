# Decisions & ADR

## Zweck der Schicht
- Nachvollziehbare Dokumentation architektonischer Entscheidungen.
- Verknüpfung von Kontext, Entscheidung und Konsequenzen.
- Historisierung von Statusänderungen bei Entscheidungen.

## Erlaubte Artefakte
- ADR-Dokumente auf Basis des ADR-Templates.
- Entscheidungsänderungen mit klarer Versionsspur.
- Referenzen auf betroffene Handbook-/Meta-Dokumente.

## Verbotene Artefakte / Anti-Patterns
- Unspezifische Meeting-Protokolle ohne Entscheidungscharakter.
- Entscheidungen ohne Kontext oder ohne Konsequenzanalyse.
- Versteckte Richtlinien außerhalb von ADR/Meta.
