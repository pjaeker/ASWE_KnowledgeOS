# Templates & Harness

## Zweck der Schicht
- Sammlung wiederverwendbarer Vorlagen.
- Bereitstellung von Prüf- und Ausführungsgerüsten.
- Operationalisierung kuratierter Wissensbausteine.

## Erlaubte Artefakte
- Template-Stubs und strukturierte Boilerplates.
- Harness-Skripte und Ausführungsanleitungen (später).
- Verweise auf zugrundeliegende Handbook-Inhalte.

## Verbotene Artefakte / Anti-Patterns
- Unstrukturierte Rohnotizen ohne Template-Bezug.
- Dauerhafte Ablage von Entscheidungsprotokollen.
- Harte Kodierung von Secrets oder Umgebungsdaten.
