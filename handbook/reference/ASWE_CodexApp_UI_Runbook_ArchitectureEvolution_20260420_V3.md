# ASWE Codex App UI Runbook - Architecture Evolution

## Vorbedingungen
- trusted project
- wirksame Projektinstruktion
- wirksame `.codex`-Flaeche
- Readiness-/Audit-Drift geprueft und als `blocking` oder `non-blocking` ausgewiesen

## Zusatzausfuehrung fuer Prompt-Architektur
Bevor aktive Projektanweisungen gepostet werden:
1. Prompt-Preflight-Regel beachten
2. falls strukturkritisch: zuerst Diagnose + Schaerfung
3. Kickoff und Thread-Start nur gemaess PromptPack_Rendering_Contract rendern

## Hinweis
Codex ist Ausfuehrungsumgebung, nicht semantischer Ursprung.
