# ASWE Codex App Bootstrap Prompt Pack

## Renderherkunft
Dieses Prompt Pack ist ein Rendering aus:
- `ASWE_PromptEval_Contract_20260420_V1.md`
- `ASWE_Prompt_Preflight_Operationalization_20260420_V1.md`
- `ASWE_CodexApp_PromptPack_Rendering_Contract_20260420_V1.md`

Es ist keine eigenstaendige Governancequelle.

## Projekt-Kickoff
```text
Arbeite im Repo ASWE_KnowledgeOS in einem bounded Prompt-Architecture-Slice unter ArchitectureEvolution.
Fuehrend sind ENTRY_LATEST, RepoStatusUpdate, aktiver Umbrella-Workstream, StartContract, PromptEval_Contract, Prompt_Preflight_Operationalization und PromptPack_Rendering_Contract.
Ziel ist die bounded Materialisierung eines selbstverbesserungsfaehigen Prompt-Unterbaus ohne neuen Primaerpfad und ohne zweite Governancequelle.
```
