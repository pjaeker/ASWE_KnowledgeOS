# Handbook

## Zweck der Schicht
- Kuratierte Wissensbasis für wiederholbare Nutzung.
- Strukturierung nach Diátaxis für unterschiedliche Lernziele.
- Konsolidierung freigegebener Inhalte aus vorherigen Schichten.

## Erlaubte Artefakte
- Inhalte in `tutorial/`, `howto/`, `reference/`, `explanation/`.
- Freigegebene und nachvollziehbar versionierte Dokumente.
- Querverweise auf Metadokumente und Entscheidungen.

## Verbotene Artefakte / Anti-Patterns
- Ungeprüfte Rohnotizen direkt aus dem Blackboard.
- Vermischung von Diátaxis-Kategorien ohne Kennzeichnung.
- Ausführbare Harness-Artefakte als Ersatz für Dokumentation.

> Hinweis: Die Diátaxis-Unterordner liegen unter `tutorial/`, `howto/`, `reference/`, `explanation/`.
