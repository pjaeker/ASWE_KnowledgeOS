# ASWE_Axiomraum_V14_schlanke_navigation_materialisierbar_20260423

## Zweck
Dieses Paket dokumentiert den aktuell verfuegbaren materialisierten Versionsverlauf des ASWE-Axiomraums duplettenfrei und repo-materialisierbar.

## Bedienreihenfolge
1. `meta/manifest.md` lesen.
2. `final/ASWE_Axiomraum_Grundlagendokument_20260423_V11.md` als aktuelles Grundlagendokument lesen.
3. `support/current/ASWE_Axiomraum_Zeilenbelegmatrix_20260423_V1.md` als aktuelle Zeilenbelegmatrix lesen.
4. `support/history/ASWE_Axiomraum_Versionschronik_20260423_V1.md` und `support/history/ASWE_Axiomraum_Deduplizierungsbericht_20260423_V1.md` fuer Verlauf und Deduplizierung lesen.
5. `meta/paket_pruefung.md` fuer Paketstatus lesen.

## Rollen
- `final/` enthaelt ausschliesslich das aktuelle Grundlagendokument.
- `support/current/` enthaelt aktuelle Pruefartefakte.
- `support/history/unique/` enthaelt eindeutige historische Markdown-Dateien ohne identische Duplikate.
- `support/package_sources/` registriert alte ZIP-Paketquellen.
- `meta/` enthaelt Paketmetadaten und Pruefsummen.
