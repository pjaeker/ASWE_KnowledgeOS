# ASWE Pro-Model Autonomous-Start Package

## Zweck
Dieses Paket ist der kuratierte Review- und Evaluationssatz fuer ein Pro-Modell, das die Sollarchitektur des
ASWE_KnowledgeOS sowie ihre spaetere operative Nutzbarkeit pruefen und weiterentwickeln soll.

## V3-Schnitt
V3 ist repo-kompatibel additiv geschnitten:
- die Pro-Modell-Unterlagen liegen gesammelt unter `meta/packages/ASWE_ProModel_AutonomousStart_20260414_V3/`
- das konkrete lokale Betriebs- und Validierungspaket wird nicht mehr als ZIP referenziert, sondern ueber
  eine Repo-Referenzdatei auf die additiv materialisierten DocumentSlice-Artefakte abgebildet

## Zwei-Paket-Logik
1. **ProModel-Paket**
   - fuer grosse Architektur-, Governance- und Reviewarbeit
2. **DocumentSlice-Repo-Artefakte**
   - fuer den konkreten bounded Evidence-Intake-Selbstevolutionsstart und dessen Evaluation

## Empfohlene Nutzung
- Zuerst dieses Paket lesen und das Reviewziel setzen.
- Danach die kanonischen 2026-04-08-Basen und Shared Governance aus dem Repo lesen.
- Erst danach die DocumentSlice-Referenz und die dort genannten additiven Runtime-Artefakte lesen.

## Paketinhalt
- Projektinstruktion fuer die Pro-Modell-Nutzung
- Master Briefing
- Gap and Decision Log
- Source Index
- Kickoff / Expected Deliverables
- Pro-Model Request
- Upload Order
- Manifest
- Repo-Referenz auf die additiven DocumentSlice-Artefakte
