# AgenticSWE-Knowledge-OS (Option A-Lite)

Dieses Repository ist ein **Minimal-Gerüst** für ein ASWE Knowledge-OS.
**Option A-Lite: Stubs only – Inhalte folgen.**

## 4-Schichtenmodell (grob)
- **Library**: Rohquellen, Evidenz und unverarbeitete Grundlagen.
- **Blackboard**: Arbeitsbereich für Hypothesen, Claims, Widersprüche und offene Fragen.
- **Handbook**: Kuratierte Wissensdarstellung nach Diátaxis.
- **Templates & Harness**: Ausführbare Bausteine, Vorlagen und Prüfgerüste.
