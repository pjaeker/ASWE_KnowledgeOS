# Glossary

## Zweck der Schicht
- Zentraler Ort für Terminologie und Begriffsabgleich.
- Reduktion von Mehrdeutigkeiten über Synonymregeln.
- Grundlage für konsistente Dokumentation in allen Schichten.

## Erlaubte Artefakte
- Begriffseinträge mit Referenz auf Kanondokumente.
- Synonym- und Alias-Hinweise mit Status.
- Änderungsnotizen für Begriffsentscheidungen.

## Verbotene Artefakte / Anti-Patterns
- Unscharfe, nicht definierte Fachbegriffe ohne TODO-Status.
- Vermischung von Glossar und Policy-Definitionen.
- Doppelpflege identischer Begriffe in mehreren Orten.
